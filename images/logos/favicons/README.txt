Pest Control Inc. favicon package

Recommended files to upload to /public or your site root:
- favicon.ico
- favicon-16x16.png
- favicon-32x32.png
- apple-touch-icon.png
- android-chrome-192x192.png
- favicon-512.png
- site.webmanifest

Also included:
- favicon-full.ico and full-logo PNG versions that preserve the full rectangular logo.

Recommended browser favicon is favicon.ico, which uses the PCI lettermark so it stays clearer at small sizes.
